## 💡 Sobre Nosotros
**Kóonex** (del maya: *"Vamos"*) es un equipo de ingenieros de software en formación con una meta clara: **profesionalizar el desarrollo estudiantil**. No nos limitamos a cumplir con requisitos académicos; construimos herramientas robustas, escalables y pensadas para resolver problemas del mundo real.

## 🏛️ Proyecto Destacado: Nexnotarial
**Nexnotarial** es nuestro proyecto insignia actual. Es una plataforma integral diseñada específicamente para la **digitalización y automatización de procesos en Notarías Públicas**.

### 🌟 Características Clave
* **Gestión de Expedientes:** Control total sobre el flujo de documentos legales.
* **Automatización de Flujos:** Reducción de tiempos manuales en trámites administrativos.
* **Arquitectura Escalable:** Implementado bajo un entorno de **Docker** con **PostgreSQL**, garantizando estabilidad y facilidad de despliegue.
* **Reportes Especializados:** Generación automática de documentos listos para firma y archivo.

> *Este proyecto demuestra nuestra capacidad para entender procesos legales complejos y traducirlos en soluciones tecnológicas limpias.*
